= Tiebreaker configuration workflow
:icons: font
:imagesdir: ../media/

[.lead]
After installing the Tiebreaker software, you can add MetroCluster configurations, configure SNMP settings, and begin monitoring your MetroCluster environment.

.About this workflow

You can use this workflow to configure the Tiebreaker software after installation.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:concept_configuring_the_tiebreaker_software.html[Configure the Tiebreaker software]
[role="quick-margin-para"]
Add MetroCluster configurations to the Tiebreaker software, import certificates, and configure monitoring settings.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:task_configure_snmp_setts_for_tiebreaker_software.html[Configure SNMP settings for Tiebreaker software]
[role="quick-margin-para"]
Configure SNMP settings so that the Tiebreaker software can send SNMP notifications for MetroCluster events.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png[Three]link:concept_monitoring_the_mcc_configuration.html[Monitor the MetroCluster configuration]
[role="quick-margin-para"]
Use the Tiebreaker software to monitor the MetroCluster configuration status, view SNMP events, and manage AutoSupport settings.
